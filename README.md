# 🌿 Virtual Herbal Garden

A web-based **Virtual Herbal Garden** that allows users to explore medicinal plants from traditional AYUSH systems such as **Ayurveda, Yoga, Unani, Siddha, and Homeopathy**.

This project provides an interactive platform where users can learn about herbal plants, their botanical information, medicinal benefits, and traditional uses.

---

## 🚀 Features

- 🌱 Explore medicinal plants
- 🔍 Search plants by name
- 🎤 Voice search using Web Speech API
- 🌍 Embedded 3D plant models (Sketchfab)
- 📚 Educational resources about herbal medicine
- 📱 Responsive design
- ✉️ Contact form

---

## 🧠 Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript**
- **Web Speech API**
- **Sketchfab 3D Model Embeds**
- **Panolens.js**

---

## 🌿 Featured Plants

The platform currently includes the following plants:

- **Aloe Vera**
  - Botanical Name: *Aloe barbadensis miller*
  - Family: Asphodelaceae
  - Benefits: Wound healing, skin care, blood sugar control

- **Tulsi (Holy Basil)**
  - Botanical Name: *Ocimum basilicum*
  - Family: Lamiaceae
  - Benefits: Immunity booster, anti-inflammatory, diabetes control

- **Neem**
  - Botanical Name: *Azadirachta indica*
  - Family: Mahogany
  - Benefits: Antibacterial, antifungal, improves skin health

- **Turmeric**
  - Botanical Name: *Curcuma longa*
  - Family: Ginger family
  - Benefits: Anti-inflammatory, digestion improvement
